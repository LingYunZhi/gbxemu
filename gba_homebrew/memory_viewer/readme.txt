Usage:
SELECT: Toggle bus width between 8, 16, 32 bit
START+SELECT: Write 32 KiB of memory from address to cartridge SRAM
START+DOWN: increase address by 1
START+UP: decrease address by 1
DOWN: increase address by 8
UP: decrease address by 8
RIGHT: increase address by 128
LEFT: decrease address by 128
A: increase address by 64 KiB
B: decrease address by 64 KiB
R: go to next memory region
L: go to previous memory region


The screen refreshes every two seconds if no command is entered.
